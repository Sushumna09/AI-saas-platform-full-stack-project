# QuickAI Studio 🚀

AI-powered SaaS platform for content generation, image creation, resume analysis, and community sharing.

QuickAI Studio is a production-ready AI SaaS platform that enables users to generate articles, blog titles, AI images, remove image backgrounds, review resumes, and share creations with the community.

Frontend: https://quick-ai-chi-gray.vercel.app

Backend: https://quick-ai-server-pied-two.vercel.app

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Sushumna09/AI-saas-platform-full-stack-project.git
cd AI-saas-platform-full-stack-project
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```
